Bienvenido a Task & Client Manager
===================================

.. raw:: html

   <p style="font-size:1.2em; color:#2D3536;">
   Sistema CRM de escritorio para la gestión de clientes, tareas y reuniones.
   </p>

**Task & Client Manager** es una aplicación de escritorio desarrollada en
Python con CustomTkinter y SQLite, diseñada para freelancers y
emprendedores que necesitan organizar su cartera de clientes de forma
sencilla y visual.

Características principales
---------------------------

- Gestión completa de clientes con datos de contacto, categoría y fuente
- CRUD de tareas por cliente con fechas y estados
- CRUD de reuniones por cliente con control de pagos
- Colores diferenciados según estado de pago
- Calendario emergente para selección de fechas
- Búsqueda y ordenamiento en tablas
- Tooltips en celdas con texto largo
- Arquitectura MVC modular
